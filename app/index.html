Usage

Using Gutenberg is very simple, a Gutenberg() function is globally exposed for you to use. You pass it two arguments: the first, an object representing an arbitrary number of key/value pairs; the second, a string representing the template that uses the Moustache like {{foo}} syntax to denote where the values corresponding to the first argument's keys should be replaced.
